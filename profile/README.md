# ğŸš€ How to Start
#### 1. Clone The Repository
```bash
git clone https://github.com/your-org/Backend.git
git clone https://github.com/your-org/Frontend.git
```

#### 2. Backend ENV Setting
```bash
# Backend/.env
SECRET_KEY=
DEBUG=

# Database
DATABASE_HOST=
DATABASE_NAME=
DATABASE_USER=
DATABASE_PASSWORD=

# Redis
REDIS_URL=

# Celery
CELERY_BROKER_URL=

# External APIs
KIS_APP_KEY=
KIS_APP_SECRET=
DART_API_KEY=
NAVER_CLIENT_ID=
NAVER_CLIENT_SECRET=
GEMINI_API_KEY=
```

#### 3. Run Docker
```bash
docker-compose up --build
```

#### 4. Frontend Install & Run
```bash
yarn install
yarn run dev
```
